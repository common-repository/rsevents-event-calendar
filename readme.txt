With RSEvents! you can create events and share them with your readers. It also has a built-in calendar that will show up your events.

== Description ==

<p>RSEvents! is the ideal WordPress plugin for everyone organizing events starting from individuals that invite close people to join a party to music bands organizing concerts or companies organizing meetings.</p>

<p>Creating an event is really easy and you get a load of options. For example you can set up recurring events repeating every day, month or year, manage the location of the event and choose wheter the event is public or private</p>

<p>RSEvents! also lets you add files to an event so you can attach brochures or entrance tickets that your readers may print.</p>


<p><a href="http://www.rsplugins.com/wordpress-plugins/wordpress-event-calendar" target="_blank"><strong>RSPlugins.com</strong></a> - Check also the pro version. Loads of features that can make your life easier when organizing events.</p>

== Installation ==

   1. Upload the rsevents-event-calendar folder to the /wp-content/plugins/  directory.
   2. Activate the plugin through the 'Plugins' menu in WordPress.

You will see the RSEvents! menu in the Wordpress Admin Sidebar.
